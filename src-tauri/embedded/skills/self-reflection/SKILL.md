# Self Reflection

Apply this loop before completing non-trivial work.

## THINK

Identify the actual user goal, constraints, risks, and the smallest useful completion state. Separate known facts from assumptions.

## WRITE

Produce the answer, plan, patch, or tool result needed to move the work forward. Keep it concrete and checkable.

## REFLECT

Before finalizing, ask:

- Did I answer the newest user request?
- Did I preserve unrelated user work?
- Did I verify claims that matter?
- Did I leave a hidden blocker or missing step?
- Is the response concise enough for the user's next action?

## REVISE

Correct gaps before handing off. If a risk remains, state it directly with the best next verification step.
